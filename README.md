# Projet-I1-JLE
Bienvenue sur le dépôt officiel du projet **Codenames Éducatif**. 
Ce jeu de cartes numérique a pour but de faire découvrir les concepts clés des **plateformes collaboratives Peer-to-Peer (P2P)** de manière ludique.

---

## Vision du Projet
L'objectif est de transformer les mécaniques du célèbre jeu *Codenames* pour enseigner les enjeux de l'économie de partage : effets de réseau, confiance numérique, désintermédiation, et modèles de revenus.

## Méthodologie de Collaboration (Notation)
Pour garantir une collaboration transparente et efficace, l'équipe suit ce workflow :

1. **GitHub Discussions :** Utilisation pour le brainstorming, les débats de game design et les décisions majeures.
2. **GitHub Projects :** Gestion des tâches via un tableau Kanban (À faire, En cours, Revue, Terminé).
3. **Issues & Branches :** Chaque nouvelle fonctionnalité commence par une *Issue*. Le développement se fait sur une branche dédiée (ex: `feature-ui-design`).
4. **Pull Requests (PR) :** Aucune modification n'est fusionnée dans `main` sans une relecture croisée. Le PM valide la cohérence globale avant le merge.
